# 时间选择器

With `MDX`, we can define a story for `DatePickers` right in the middle of our
Markdown documentation.

export const Template = (args, { argTypes }) => ({
  props: Object.keys(argTypes),
  components: { DatePickers },
  template: '<DatePickers v-bind="$props" />',
});

## 参数
+ 时间段：value ['yyyy-MM-dd','yyyy-MM-dd']
+ 时间格式：format 默认为"yyyy-MM-dd"，可选项"tiemstamp",时间戳还是日期格式
+ 提示语： placeholder
+ 是否默认：default 默认为true,true：默认值,false:不选择默认选项，前三个参数可用
