# Input

With `MDX`, we can define a story for `Input` right in the middle of our
Markdown documentation.

export const Template = (args, { argTypes }) => ({
  props: Object.keys(argTypes),
  components: { Input },
  template: '<Input v-bind="$props" />',
});

## 参数
+ 类型：type 
+ 是否清除：clearable
+ 是否禁用：disabled
+ 提示语： placeholder
