# storybook-demo

## Project setup

### 安装依赖

```
npm install
```
### 可以根据目前所处理的应用程序的不同部分来决定同时运行下述中的一个或者多个。

### 开发环境运行
```
# 运行storybook
npm run storybook

# 运行Vue
npm run serve
```

### 打包生产环境代码
```
npm run build-storybook

npm run build
```
